## 🧠 Overview

This project is a Generative AI-powered video summarizer that converts long-form video content into short, readable summaries.

It works by:

1. Extracting audio from a YouTube video
2. Transcribing speech using OpenAI Whisper
3. Splitting long text into chunks
4. Generating summaries using transformer models

---

## ⚙️ Tech Stack

* Python
* OpenAI Whisper (Speech-to-Text)
* Hugging Face Transformers
* Gradio (UI)
* yt-dlp (YouTube audio extraction)
